Hundreds of people (especially children) go missing every day in India.  
This project uses AI to help authorities track and identify missing persons faster.

---

## Project Overview
This AI-assisted platform helps locate missing individuals by:

- Facial recognition using MediaPipe Face Mesh
- Scanning large volumes of images (CCTV, public uploads)
- Web and mobile interfaces for reporting and monitoring

---

## Key Features
- Register missing person cases with image upload  
- Automated facial feature extraction  
- Fast image matching  
- Admin + public dashboards  
- SQLite lightweight database
